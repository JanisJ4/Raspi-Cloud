Raspi-Cloud: Unlimited Data Privacy for Truly Everyone

Secure cloud solutions keeping personal data on your own server.

Why Raspi-Cloud?

Raspi-Cloud revolutionizes the way you think about cloud storage. Designed to run smoothly on low-powered devices like the Raspberry Pi, it ensures that anyone can set up their own cloud server, offering an unparalleled level of data privacy and security. With Raspi-Cloud, your data stays at home, under your control, and is accessible even over slow internet connections, making it truly location-independent.

This cloud solution is for those who value their privacy and seek a reliable, cost-effective way to manage their data independently. It's built to be lightweight, yet powerful enough to handle your storage needs without requiring expensive hardware or fast internet speeds.

Installation

Follow these steps to get Raspi-Cloud up and running on your Raspberry Pi.

1. Cloning the Repository

Start by updating your package list and installing git:

sudo apt-get update
sudo apt-get install -y git

Then, clone the Raspi-Cloud repository:

git clone https://github.com/JanisJ4/Raspi-Cloud.git

2. Setting Up the Cloud

Navigate to the Raspi-Cloud directory and run the installation script. Choose HTTPS for a secure setup:

cd Raspi-Cloud 
sudo bash install_script_https.sh

For HTTP (not recommended), use:

sudo bash install_script_http.sh

3. Accessing the Web Interface

Once Raspi-Cloud is installed, you can access the web interface through your browser. Simply enter the local IP address of your Raspberry Pi followed by the port number used by Raspi-Cloud: 

raspberrypi.local

Replace "raspberrypi.local" with the actual IP address or Domain if necessary. The default login credentials will be provided during the installation process.
